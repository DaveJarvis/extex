#ExTeX() Testing

#TeX() is accompanied by the trip test. This test tries to ensure that any system named &TeX; works properly. This means that it is a regression test suite.

#ExTeX() needs a regression test suite which should ensure

  • that #ExTeX() properly provides the functionality of #TeX() for backward compatibility.
  • that the functionality of #ExTeX() stays intact across versions of #ExTeX()

Some test cases and a framework for testing is already present in #ExTeX(). This is based on JUnit.

Wanted

Several persons are wanted to cover different aspects of this important and time consuming tasks:

  • Someone with skills in #TeX() is required to write a bunch of additional test cases.
  • Someone with skills in Java and #TeX() to code the test cases in JUnit.
  • Someone with skills in Java to regularly run the test cases and assign the problematic cases to maintainers of single pieces of code.
  • Maybe someone with skills in Maven to automate the process.
