εχTeX Site

Thus module contains the site builder and the source file for the εχTeX site.

Creating the εχTeX Site

The εχTeX site can be created in this directory with the help of Maven3:

# mvn compile

This command requires that Maven is properly installed on your system and the executable can be found on the search path.

The progress log appears on the standard output:

After a successful completion of the command the site can be found as directory target/www.

If the build fails the verbose log messages of Maven can be used to determine the problem. Usually just reading the messages leads you into the right direction.
