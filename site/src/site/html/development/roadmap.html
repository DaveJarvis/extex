Road Map

The releases of #ExTeX() will be driven by features contained. The list below shows some of the short term features to be provided.

Since the development of #ExTeX() is a spare-time activity of the #ExTeX() team no dates can be confirmed for the releases.

Releases

Release #TeX() Compatibility

The release is feature complete for the #TeX() compatibility mode. This means that all primitives of #TeX() are present and functional. The typesetting can produce a dvi file which is equivalent – but not necessarily identical – to the file produced by #TeX().

Release #eTeX() Compatibility

The release is feature complete for the #eTeX() compatibility mode. This means that all primitives of #eTeX() are present and functional. The typesetting can produce a dvi file which is equivalent – but not necessarily identical – to the file produced by #eTeX().
