Maintaining the Web Pages

The public relations work is rather important for any Open Source project. It is necessary to reflect the current activities and the progress to attract attention.

#ExTeX() has some Web pages and a simple framework for their generation which need to be updated regularly.

The Web pages are generated from simple sources – mainly HTML. This allows to separate the content from the layout. The building engine is written in Java. It uses Apache Velocity as templating engine.

Wanted

  • Someone with skills in writing and HTML is needed to maintain the Web pages.
