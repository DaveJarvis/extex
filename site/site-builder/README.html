The εχTeX Site Builder

The site for εχTeX is created from a set of sources. This creation utilizes Apache velocity as templating engine.

The following kind of files are created:

  • The HTML files
  • The site map
  • The RSS files

Creating the Site

The εχTeX site can be created in this directory with the help of Maven:

  # mvn package

This command requires that Maven3 is properly installed on your system and the executable can be found on the search path.

After a successful completion of the command the site can be found in the directory target/site.

If the build fails the verbose log messages of Maven can be used to determine the problem. Usually just reading the messages leads you into the right direction.

Organization of the Files

The sources of the site can be found in src/site.
