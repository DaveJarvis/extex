Repository

The sources are publicly available. For this purpose there is a Subversion repository hosted by BerliOS:

http://svn.berlios.de/svnroot/repos/extex/trunk

The following sample shows how to check-out the complete sources of #ExTeX() on a Unix box (or in Cygwin on Windows): 

 % svn checkout http://svn.berlios.de/svnroot/repos/extex/trunk

There are many other tools around to access a Subversion repository. Some of them are:

  • The Subclipse plug-in for Eclipse.
  • TortoiseSVN for the Windows Explorer.

The description of those tools is beyond the scope of this introduction.

The committing is restricted to members of the #ExTeX() team!

If you are interested to become a member of the #ExTeX() team write to extex@dante.de.
