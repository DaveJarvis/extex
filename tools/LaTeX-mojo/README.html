The LaTeX Mojo

The LaTeX mojo tries to create a document from its source files.

Compiling the LaTeX Mojo

The LaTeX mojo can be created in this directory with the help of Maven2: 

  # mvn package

This command requires that Maven2 is properly installed on your system and the executable can be found on the search path.

After a successful completion of the command the jar can be found as the file target/maven-latex-plugin-1.0-SNAPSHOT.jar.

If the build fails the verbose log messages of Maven can be used to determine the problem. Usually just reading the messages leads you into the right direction.
