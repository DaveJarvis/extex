εχTeX Maven Skin

The εχTeX Maven Skin provides a Maven skin for generating a Maven site with the maven-site-plugin. All artifacts and scripts needed are contained.

Installing εχTeX Maven Skin

εχTeX Maven Skin can be installed in the local repository with the help of Maven: 

  # mvn install

This command requires that Maven is properly installed on your system and the executable can be found on the search path.
