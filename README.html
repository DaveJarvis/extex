εχTeX

εχTeX aims at the development of a high quality typesetting system. This development is massively based on the experiences with the typesetting system TeX. Despite of its age TeX can still be considered a very good choice. Nevertheless design decisions which where reasonable at the time of the writing of TeX can nowadays no longer be considered as state of the art.

The stability of TeX is one of its virtues. On the other side it makes it hard to improve the system – even in the few areas which deserve improvements. The new millenium needs a system which is open for further development and at the same time compatible with TeX as much as reasonable.

εχTeX on the Web

εχTeX is present on the Web with the Web site at www.extex.org. This Web site is hosted by DANTE e.V.

The source for this site can be found in the directory www. The site is regenerated from these source each night.

Getting Started – with Reading

If you are interested in using εχTeX you should have a look at the εχTeX User's Guide. The source code for the user's guide can be found in the directory doc/UsersGuide.

If you are interested in the sources you should have a look at the Developer's Guide. The source code can be found in doc/DevelopersGuide. A compiled version can be downloaded from the εχTeX web site.

Licenses – Legal Issues

εχTeX is developed under an Open Source license. It is meant to be free. Thus the GNU Library General Public License has been chosen.

εχTeX uses some libraries. They usually come with a license of their own. Thus watch out to respect those licenses as well.

εχTeX contains some documentation. It is distributed under the GNU Free Documentation License.

File Organization

CLI
contains a general purpose library for command line parsing and processing.

ExBib
εχTeX has a sub-project εχBib which aims at providing a bibliography processor. εχBib can be found in the sub-directory ExBib.

ExIndex
εχTeX has a sub-project εχIndex which aims at providing an index processor. εχIndex can be found in the sub-directory ExIndex.

ExTeX
εχTeX has the primary goal to provide a typesetter in the spirit of TeX. This directory contains the specific modules for this program.

ExTeX-resource
contains a library for resource loading and several supporting functionality for components and configuration.

Sandbox
contains sand boxes of developers. Here code can be put under version control without integrate it into the official project tree.

build
contains the Maven-based build system. It also contains some fragments from the old Ant-based build system which is now outdated, does not work any more, and will sooner or later be overcome completely.

develop
contains configurations and tools for the development.

doc
contains documentation for the overall project and εχTeX itself. Additional documentation can be found in the sub-projects.

library
contains the library of collected papers and documents.

maven2
contains Maven projects used in εχTeX which do not have a public site of their own. The contents is directly published each night under the URL http://www.extex.org/maven2/.

site
contains the relaunch of the web pages on www.extex.org. It is not active yet.

src contains additional sources. Thus we follow the conventions of Maven.

target
is not contained in the repository. It is created on the fly during compilation to receive the intermediate files and results.

texmf
contains sources for a texmf tree.

tools
contains some tools.

util
contains some utilities. Usually they are shell scripts or programs of some sort which do not need the global build system of εχTeX.

www
contains the sources for the web pages for www.extex.org.

Compiling εχTeX or Sub-Projects

εχTeX is organized into the sub-projects ExTeX, ExBib, and ExIndex. These sub-projects are organized in a way that they can be compiled independently. Consult the ReadMe files in the sub-projects to get instructions to compile them.

εχTeX utilizes Maven for compilation.
