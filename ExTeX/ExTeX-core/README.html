The εχTeX Core

εχTeX is a typesetter in the spirit of TeX. The core component is contained in this module. The core component provides the definition of registers and some fundamental support classes.

Compiling and Packaging the εχTeX Core

The εχTeX core can be created in this directory with the help of Maven: 

  # mvn package

This command requires that Maven is properly installed on your system and the executable can be found on the search path.
