εχTeX resource

εχTeX resource is a base component which contains various helpfull classes for deading with compoennts.

Compiling εχTeX resource

εχTeX resource can be created in this directory with the help of Maven2:

  # mvn package

This command requires that Maven2 is properly installed on your system and the executable can be found on the search path.
