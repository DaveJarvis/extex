The εχBib Installer

εχBib is a bibliography processor in the spirit of BIBTeX. εχBib is delivered in form of an installer. This installer is created with the help of IzPack.

IzPack is included in the folder src/izPack-install-* since currently no reliable Maven2 package for IzPack could be found.

Creating the εχBib Installer

The εχBib installer can be created in this directory with the help of Maven2:

  # mvn package

This command requires that Maven2 is properly installed on your system and the executable can be found on the search path.

After a successful completion of the command the installer can be found as the file target/ExBib-setup-0.1.jar. This installer includes everything needed to get started. It just requires a proper Java installation (Java 5 or higher).

If the build fails the verbose log messages of Maven can be used to determine the problem. Usually just reading the messages leads you into the right direction.
