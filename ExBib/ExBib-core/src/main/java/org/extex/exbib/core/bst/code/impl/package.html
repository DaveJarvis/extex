This package contains the classes for the built-in functions of the processor.

Built-in functions are implemented as classes which are characterized as follows:

  • They have a method execute() which performs the actions required.
  • They have a constructor which takes a String argument which is the name under which this function is registered in the processor context.

I.e. they implement the interface {@link org.extex.exbib.core.bst.Code Code}.

The functions are either integrated in the processor statically like in {@link org.extex.exbib.core.bst.Processor099Impl Processor099Impl}. Alternatively they can be loaded dynamically according to a configuration file like in {@link org.extex.exbib.core.bst.ProcessorImpl ProcessorImpl}.
