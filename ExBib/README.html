εχBib

εχBib is a bibliography processor in the spirit of BIBTeX and BIBTeX 8. It is meant to be usable as a plug-in replacement for those.

In addition several attempts have been made to make it highly configurable.

It is meant to be in line with εχTeX. Eventually it can be integrated into εχTeX or other tools. Thus the functionality of the command line program is separated from the core functionality.

The εχBib Installer

εχBib is delivered in form of an installer. This installer is created with the help of IzPack.

The εχBib installer can be created in this directory with the help of Maven2: 

  # cd ..
  # mvn package

This command requires that Maven2 is properly installed on your system and the executable can be found on the serach path.

After a succesful completion of the command the installer can be found as the file ExBib-Installer/target/ExBib-setup.jar. This installer includes everything needed to get started. It just requires a proper Java installation (Java 5 or higher).

If the build fails the verbose log messages of Maven can be used to determine the problem. Usually just reading the messages leads you into the right direction.
