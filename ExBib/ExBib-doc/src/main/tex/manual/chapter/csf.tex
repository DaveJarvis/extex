\section{CSF}%
\label{sec:csf}%
\index{csf|(}

One problem of the original \BibTeX~0.99c is the restriction to 7 bit
characters. To solve this problem the cs files have been introduced in
\BibTeX~8\index{BibTeX 8|\BibTeX 8}:
\IM{8x}

\begin{verbatim}
  sort=csf
\end{verbatim}

\begin{verbatim}
  sort=csf:resource
\end{verbatim}

\INCOMPLETE

\index{csf|)}

\endinput
